metadata_fetch

Description:
A small exercise of retrieving video information using ffmpeg libraries in C
for better understanding of the functionality.

Requirements:
You must have ffmpeg installed.
And a computer that has a terminal command program.

Instructions:
1. Type make on the terminal
2. Type "./metadata_fetch video_name.mov"

Expected Output:
Loading: /home/(user)/(directory)/video_name.mov
Frame rate: 30.000
Timecode Mode: Non Drop Frame
Width: 3840
Height: 2160
Duration: 00:10:34:16 seconds 

or
Loading: /home/(user)/(directory)/video_name.mov
Frame rate: 59.940
Timecode Mode: Drop Frame
Width: 1920
Height: 1080
Duration: 00:02:00;00 seconds
